# Math Table JavaScript Calculator

## Overview
This repository contains a web-based calculator application developed as part of my coursework while pursuing a Bachelor of Business Administration in Information Systems and Technology (IST).

The calculator allows users to perform arithmetic calculations using both on-screen controls and keyboard shortcuts. The project demonstrates JavaScript programming concepts including event handling, mathematical expression evaluation, calculation history management, keyboard input processing, and dynamic user interface functionality.

## Features
* Interactive calculator interface
* Arithmetic expression evaluation
* Keyboard and mouse input support
* Adjustable decimal precision settings
* Previous calculation recall
* Backspace and delete functionality
* Dynamic calculation history
* Real-time expression processing
* Calculator instruction panel

## Technologies Used
* HTML5
* CSS3
* JavaScript
* DOM Manipulation
* Event Handling

## Project Files
* `mt_calc_txt.html` – Main calculator webpage
* `mt_calc_txt.js` – Calculator logic and event handling
* `mt_base.css` – Base stylesheet
* `mt_styles.css` – Website layout and styling
* `mt_calc.css` – Calculator-specific styling
* Calculator branding and image assets
